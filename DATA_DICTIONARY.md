# Data Dictionary

## Dataset: NYC 311 Complaints

### Columns

- **unique_key**
  - Description: Unique identifier for each complaint
  - Type: Integer

- **created_date**
  - Description: Date and time the complaint was created
  - Type: Datetime

- **agency**
  - Description: Agency responsible for handling the complaint
  - Type: String

- **complaint_type**
  - Description: Type/category of complaint
  - Type: String

- **descriptor**
  - Description: More detailed description of the complaint
  - Type: String

- **location_type**
  - Description: Type of location where complaint occurred
  - Type: String

- **incident_zip**
  - Description: ZIP code of incident location
  - Type: String

- **city**
  - Description: City where complaint occurred
  - Type: String

- **status**
  - Description: Current status of complaint
  - Type: String

- **resolution_description**
  - Description: How the complaint was resolved
  - Type: String
